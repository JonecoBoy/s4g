# Welcome to S4G

**S4G** (Super Simple Static Site Generator) is a fast, pluggable SSG built in Go.

## Features

- 🔌 **Pluggable data sources** — Markdown today, SQL/JSON/REST tomorrow
- 🎨 **Go html/template** rendering with a layout + page system
- ⚡ **Bubbletea TUI** build summary

## Getting Started

```bash
s4g build
```

That's it. Your site will be generated in `dist/`.
